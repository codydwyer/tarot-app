# tarot-app
This is yarn monorepo containing a fullstack tarot app.

The high-level purpose of this repo is to keep my web dev skills fresh.

This app has three main functionalities:
  * Allow users to pull a spread of tarot cards
  * Allow users to build custom decks
  * Allow users to build custom spreads

## Getting started

### Start dev packages

```
// install dependencies
yarn

// launch all dev servers
yarn dev
```
